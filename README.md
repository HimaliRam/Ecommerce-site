# 🛍️ ShopEase – E-commerce Web Application

ShopEase is a full-stack e-commerce web application built using Angular (Frontend), .NET (Backend), and MS SQL Server (Database). This project focuses on providing a smooth user shopping experience with essential e-commerce features.

---

## 🚀 Features

* 🔐 User Authentication (Login, Signup, Forgot Password)
* 🛒 Add to Cart & Buy Now
* ❤️ Wishlist Management
* 🔍 Product Search & Filter
* 📦 Product Listing by Categories
* 💾 Data stored securely in database
* 🔄 Real-time UI updates (cart & wishlist)
* 📱 Responsive UI design

---

## 🛠️ Tech Stack

* **Frontend:** Angular
* **Backend:** .NET Web API
* **Database:** MS SQL Server
* **Styling:** CSS + Bootstrap

---

## 📂 Project Structure

* `/Frontend` → Angular Application
* `/Backend` → .NET API
* `/Database` → SQL scripts

---


## 🌟 Key Highlights

* Fully functional **user-side e-commerce system**
* Clean UI with smooth navigation
* Proper database integration
* Scalable project structure

---

## 📌 Note

This project is focused on **user-side functionality only** (no admin panel).
